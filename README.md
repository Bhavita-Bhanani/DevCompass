# DevCompass

DevCompass is a developer resource hub designed to help students and aspiring developers navigate their learning journey.

## Features

- Curated learning platforms
  - CS50
  - freeCodeCamp
  - MDN Docs
  - W3Schools
  - GeeksforGeeks

- Developer roadmaps
  - Frontend Development
  - Backend Development
  - Python Development
  - AI / ML
  - Cybersecurity
  - Cloud Computing

- Coding practice resources
  - LeetCode
  - HackerRank
  - Codeforces
  - CodeChef

- Jobs & internships
  - LinkedIn Jobs
  - Internshala
  - Wellfound
  - Indeed

## Tech Stack

- HTML5
- CSS3

## Highlights

- Responsive design
- Modern green-and-white UI
- Smooth animations and hover effects
- Beginner-friendly navigation
- No frameworks or JavaScript used
